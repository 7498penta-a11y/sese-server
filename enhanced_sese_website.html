よくある質問

seseサーバーに関するよくある質問とその回答をまとめました。 他に疑問がある場合は、お気軽にDiscordでお聞きください。

💰 参加費用はかかりますか？ +

完全無料です！Discordサーバーへの参加、マインクラフトサーバーでのプレイ、 すべてのイベント参加に費用は一切かかりません。任意の寄付システムはありますが、 強制ではありません。

🎮 どのバージョンのマインクラフトが必要ですか？ +

Java Edition 1.20.x 最新版を推奨していますが、1.19.x以降であれば参加可能です。 統合版（PE/Switch/Xbox/PS4）の方もGeyserを通じて参加できます。 詳細な接続方法はDiscordで案内しています。

👶 年齢制限はありますか？ +

基本的に年齢制限はありませんが、Discord利用規約に従い13歳以上を推奨しています。 未成年の方は保護者の方の承諾を得てから参加してください。 年齢に関係なく、すべてのメンバーが快適に過ごせる環境づくりを心がけています。

🛡️ ルール違反をした場合はどうなりますか？ +

違反の程度により、注意 → 警告 → 一時停止 → 永久BANの段階的な措置を取ります。 重大な違反（荒らし、嫌がらせ、不適切な発言など）の場合は即座にBANとなることもあります。 詳細なルールはDiscordサーバー内で確認できます。

🏗️ 建築物は保護されますか？ +

はい！土地保護システムにより、あなたの建築物は完全に保護されます。 他のプレイヤーは許可なく建築物を破壊・変更できません。 また、定期的な自動バックアップにより、万が一の事態にも対応可能です。

🎁 特典やボーナスはありますか？ +

アクティブメンバーには様々な特典があります！ ログイン日数ボーナス、イベント参加特典、ランクアップ報酬、 月間MVP選出、限定アイテム配布など。 Discord Nitroブーストしてくれた方には特別な謝礼もあります。

📱 スマートフォンからでも参加できますか？ +

はい！Discordはスマホアプリがあるのでいつでもチャットに参加できます。 マインクラフトも統合版（Pocket Edition）から接続可能です。 ただし、一部機能はPC版の方が充実しているため、可能であればPCでの参加を推奨します。

🔧 技術的な問題が発生した場合は？ +

技術サポートチャンネルで24時間サポートを行っています。 接続問題、ゲーム内のバグ、Discord機能の使い方など、 どんな問題でもお気軽にご相談ください。 専門スタッフが迅速に対応いたします。
